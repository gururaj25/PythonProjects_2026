# GitHub AutoSync Pro

A production-grade desktop application for automatically syncing local folders/projects directly to GitHub repositories.

## Features

* One-click GitHub upload
* Auto Git initialization
* Automatic commit & push
* Smart sync engine
* Secure token storage
* Real-time logs
* Multi-threaded upload
* Dark mode UI
* Watchdog file monitoring

## Installation

pip install -r requirements.txt

## Run

python app.py

## Build EXE

pyinstaller --onefile --windowed app.py
