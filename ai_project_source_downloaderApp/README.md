# AI Project Source Downloader & Reconstructor

Production-style starter scaffold for:
- AI response parsing
- Multi-file reconstruction
- ZIP export
- Project structure creation
- Browser extension integration

## Stack
- FastAPI backend
- React frontend scaffold
- Python parser engine
- Chrome extension

## Run Backend
```bash
pip install -r requirements.txt
uvicorn backend.main:app --reload
```

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```
